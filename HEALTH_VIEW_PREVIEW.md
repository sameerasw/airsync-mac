## Data Display Rules

### Show Card If:
- Value is not nil
- Value > 0 (for most metrics)
- Always show (even if 0) for steps, calories

### Hide Card If:
- Value is nil
- No data available for that metric

### Progress Bar Shows If:
- Metric has a goal (steps, calories, sleep)
- Progress = value / goal
- Clamped to 0.0 - 1.0

### Format Rules
- **Steps**: Integer with comma separator
- **Calories**: Integer with comma separator
- **Distance**: 1 decimal place + "km"
- **Heart Rate**: Integer + "bpm" or "--"
- **Sleep**: "Xh Ym" format
- **Active**: Integer + "minutes"
